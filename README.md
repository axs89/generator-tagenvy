# generator-tagenvy

A generator for [Yeoman](http://yeoman.io).


## How to use

```
$ npm install -g yo
```

To install generator-tagenvy from npm, run:

```
$ npm install -g generator-tagenvy
```

Finally, initiate the generator:

```
$ yo tagenvy
```

## Testing the generator

To run the unit tests:

```
$ npm test
```
